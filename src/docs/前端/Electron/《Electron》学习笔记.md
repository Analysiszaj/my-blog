### 1.1 什么时electron?

Electron是一个使用 JavaScript、HTML 和 CSS 构建桌面应用程序的框架。 内嵌入**Chromium** 和**Node.js**， 可以使开发者能使用前端技术栈构建跨平台的桌面应用，而不需要本地开发经验
